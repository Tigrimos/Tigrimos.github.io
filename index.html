Zero Trust

Security That's Not an Afterthought

Every AI operation runs inside an isolated Ubuntu sandbox. Your host system is protected by default.

Full Sandbox Isolation

macOS: Apple Virtualization.framework VM. Windows: WSL2 distro. AI code physically cannot access your host.

Invisible by Default

Your files are completely hidden from the AI. Share only what you choose, with read-only as the default permission.

NAT Networking

The sandbox is isolated from your host network. No unexpected outbound connections from AI processes.

No Docker Required

Native OS-level virtualization on both platforms. Lighter, faster, and no dependency on Docker daemon or containers.

Security Layer macOS Windows
Sandbox Ubuntu VM via Virtualization.framework Ubuntu via WSL2
File System Host files invisible by default Host files invisible by default
Shared Folders VirtioFS opt-in, read-only default Symlink opt-in via app UI
Network NAT — VM isolated WSL2 NAT — isolated
Process Isolation VM cannot see host processes WSL2 isolated from Windows
